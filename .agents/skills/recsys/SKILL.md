This skill is pointer to recommendation resources:

- [inspecting_data.md](inspecting_data.md): techniques for inspecting and understanding the data, including exploratory data analysis (EDA), visualization methods, and statistical summaries to identify patterns, trends, and potential issues in the dataset.
- [movie_recommender.md](movie_recommender.md): building a movie recommendation system using collaborative filtering techniques, covering data preparation, model training, and generating recommendations based on user-item interactions.
- [item-based_filter.md](item-based_filter.md): implementation mechanics of item-based collaborative filtering, including data preparation, similarity calculation, and generating recommendations based on item similarities.
- [recommendations_svd.md](recommendations_svd.md): building a recommendation engine using Singular Value Decomposition (SVD) with the surprise library, covering data preparation, model training, and generating top-N recommendations for users.
- [model_performance.md](model_performance.md): evaluating the performance of recommendation models using various metrics, including precision, recall, F1-score, and mean average precision (MAP), as well as techniques for cross-validation and hyperparameter tuning.
- [weighted_hybrid_movie_recommender.md](weighted_hybrid_movie_recommender.md): implementation mechanics of a weighted hybrid movie recommender system, combining content-based and collaborative filtering approaches, including data preparation, score normalization, and generating hybrid recommendations based on a weighted formula.
