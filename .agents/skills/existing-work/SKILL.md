This skill is a pointer to three repository syntheses:

- `full_summary.md`: theory-focused causal recommendation synthesis, including causal formalisms, bias types, additive disentanglement models, dataset and metric taxonomy, graph/backbone summaries, and open problems.
- `existing_technical_synthesis.md`: code-first audit-derived comparison of prior methods, covering runtime contracts, data/graph inputs, encoder/kernel families, loss/score mechanics, evaluation protocols, performance warnings, and minimal carryover components.
- `datasets_information.md`: dataset-specific information, including files, columns, and data types for each dataset used in the repository.

Use this skill when you want a quick reference to which source contains theory versus implementation mechanics.
